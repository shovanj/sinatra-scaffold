Simple sinatra extension for datamapper that displays records  in admin panel.

h1. Usage:

Specify the columns that you want to display in the admin panel inside the configure_scaffold block.

===============================================================
configure_scaffold do
  scaffold :models => [:user, :expense] 
end

Sinatra::Base.register Sinatra::Scaffold
================================================================



To copy the view, css, js files run the following command. 

$scaffold-gen setup_files                      

The generated view files are in haml format.

That will generate the following routes 
- /admin/users
- /admin/expenses
